## Description

This repository contains the code for Project 2-1: A real-time eye tracking system with interactive web-based environments.

## Prerequisities
- **Python 3.8+**
- **Webcam**
- **pip** package manager

## Installation and Run instructions:

### Step 1: Clone/Download Repository

### Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```
### Step 3: Run the System
- Using Terminal:
```bash
cd src
python main.py
```
- Or just run main.py file thru editor.

At first, it will show a blank screen. Wait a few seconds for Eye Tracking to run.

### Step 4: Run the Interface

- **Final Product**: web/html/3D-scene3.html

Double-click the HTML file or open with `Live Server` extension in VS Code.


## Keyboard Controls in Preview

   - `1` - Linearity Horizontal test mode
   - `2` - Linearity Vertical test mode
   - `3` - Distance Sensitivity test mode
   - `D` - Cycle distance setting (close/medium/far)
   - `SPACE` - Start/Stop recording trial
   - `E` - Export logs to JSON/CSV
   - `C` - Clear all samples
   - `ESC` - Quit
